## Présentation

Ce projet propose un tableau de bord Tableau consacré à la veille stratégique autour des technologies IA, data et automatisation. Il vise à offrir une vue synthétique de type exécutif pour suivre un portefeuille de technologies, identifier les opportunités de pilotage, comparer la valeur métier à l’effort d’implémentation et visualiser les recommandations associées.

Le dashboard combine indicateurs clés, analyse de priorisation, répartition par catégorie, suivi des recommandations et exploration par domaine métier.

## Objectifs

Le tableau de bord répond à cinq questions opérationnelles :

1. Quelles technologies IA et data sont actuellement suivies ?
2. Quelles technologies méritent un pilote ?
3. Quelles technologies présentent une forte valeur métier par rapport à leur effort d’implémentation ?
4. Quelles catégories et quels domaines métier sont les plus représentés ?
5. Comment le portefeuille technologique se répartit-il entre adoption, pilote, étude approfondie et veille ?

---

## Indicateurs clés

Le dashboard suit notamment :

- le nombre de technologies suivies 
- les pilotes recommandés 
- les candidats à l’adoption 
- les technologies nécessitant une étude approfondie 
- les technologies en veille 
- les domaines IA / data couverts.

---

## Visualisations principales

### Valeur métier vs effort d’implémentation

Le nuage de points positionne les technologies selon leur valeur métier et leur effort d’implémentation. Il sert de matrice de priorisation pour distinguer les opportunités rapides, les technologies prometteuses mais plus complexes, et les sujets à maintenir en observation.

### Technologies par catégorie

Un graphique en barres horizontales montre la distribution des technologies par catégorie, par exemple Data Engineering, Framework Agentique, Modèle Fondation, BI Augmentée ou MLOps.

### Répartition des recommandations

La visualisation des recommandations synthétise l’état du portefeuille selon quatre statuts :

- Adoption ;
- Pilote ;
- Étude approfondie ;
- Veille.

### Répartition par domaine métier

Le graphique à bulles visualise la couverture des domaines métier et permet d’identifier les zones les plus représentées dans la veille.

### Top technologies par valeur métier

Un classement met en évidence les technologies présentant le plus fort potentiel métier afin de faciliter une lecture rapide du portefeuille.

---

## Méthodologie

Chaque signal de veille est structuré autour d’un schéma commun comprenant notamment :

- le nom de la technologie 
- le fournisseur 
- la catégorie 
- le domaine métier 
- le cas d’usage 
- la valeur métier 
- l’effort d’implémentation 
- la maturité 
- les indicateurs de coût et de risque 
- la priorité stratégique 
- la recommandation 
- une note analyste.

Ces champs alimentent à la fois les KPI, les visualisations descriptives et les vues de priorisation.
